üßÆ Tax Calculation Analysis

Sistem saat ini menambahkan PPN 11% secara otomatis. Ini mungkin tidak diperlukan untuk semua kasus.

// Current Tax Calculation (Automatic)
const packagePrice = 30000000;
const tax = Math.round(packagePrice * 0.11); // Always adds 11%
const total = packagePrice + tax;

// Result:
packagePrice: 30,000,000
tax: 3,300,000 // 11% of 30M
total: 33,300,000 // 30M + 3.3M = 33.3M

// Enhanced Calculation (Conditional)
let tax = 0;
let total = packagePrice;

if (!paymentData.skipTax) {
tax = Math.round(packagePrice * 0.11);
total = packagePrice + tax;
}

// Result (with skipTax = true):
packagePrice: 30,000,000
tax: 0 // Skipped
total: 30,000,000 // Correct price
